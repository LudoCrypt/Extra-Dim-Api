# Extra Dim Api
Add extra and extra Dimensions!

# Use
build.gradle ->
```groovy
repositories {
	maven {
		url 'https://jitpack.io'
	}
}

dependencies {
	modImplementation "com.github.LudoCrypt:Extra-Dim-Api:${project.exdimapi_version}"
}
```

gradle.properties ->
```groovy
exdimapi_version=1.0.1
```
